# 俳句/短歌のAIサイト(改)

俳句/短歌のAIサイトの改訂版です.
初版をNext.js/TS仕様にアップデートした上, 認証機能を追加しました.
Stripeによるマネタイズに関しては, KL渡航期などと重なり, うまく実装できませんでした.


## 機能

フォームに俳句・短歌を入力してボタンを押すとAIが感想を返す


## 技術スタック

- **バックエンド**: Nest.js (v15.1.0) App Router
- **TypeScript** - 型安全性
- **Tailwind CSS** - スタイリング
- **パッケージ管理**: npm
- **ホスティング**: Vercel
- **開発ツール**: v0, Cursor
